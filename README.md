# **Lux web page**

## 1. Set up

Download and install [Node JS](https://nodejs.org).

## 2. Clone repo

```bash
git clone https://github.com/alberti-tu/lux
```

## 2. Install required dependences

```bash
cd lux
npm install
```

## 3. Launch a development server

```bash
npm start
```

## 4. Build project

```bash
npm run build
```
